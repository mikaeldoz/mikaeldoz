## 🎯 Executive Summary

> Desenvolvedor Backend focado na construção de sistemas de alta disponibilidade e arquiteturas escaláveis. Minha especialidade reside em transformar lógicas complexas de negócios em APIs resilientes, utilizando **Java** e **Node.js**, com forte ênfase em modelagem e otimização de bancos de dados relacionais (**PostgreSQL**).

**Foco Atual:** Arquitetura de Microsserviços · Otimização de Queries em SQL · Conteinerização de Ambientes (Docker).
